# Wrighter
Web scraping/browser automation framework built on [Playwright](https://playwright.dev/python/).
# Features
- sync and async API
- plugin system
# Installation
#### From PyPi
```
pip install wrighter
```
#### From source
```
pip install git+https://github.com/zigai/wrighter.git
```
